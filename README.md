# FaunaDB のGraphQLサポート検証用のレポジトリ

## FaunaDB とは

公式

https://fauna.com

説明

- https://www.infoq.com/jp/news/2017/04/faunadb/

## 使い方

1. https://fauna.com で会員登録する
2. 適当な名前で DB を作成
3. GraphQL のタブを選択して、このレポジトリの中の schema.graphql を選択
